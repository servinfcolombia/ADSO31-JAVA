=== INSTRUCCIONES PARA COPIA DE SEGURIDAD EN GITHUB ===

=== VERIFICAR QUE GIT ESTE CONFIGURADO ===
git config --global user.name "Tu Nombre"
git config --global user.email "tuemail@example.com"

1. Abre la terminal en la carpeta de tu proyecto.
2. Inicializa el repositorio local:
   git init
3. Agrega todos los archivos al área de preparación:
   git add .
4. Realiza un commit con un mensaje:
   git commit -m "Agregando archivos iniciales"
5. Agrega el repositorio remoto de GitHub:
   git remote add origin https://github.com/servinfcolombia/ADSO31-JAVA.git
6. Sube los archivos a la rama principal (usa main o master según corresponda):
   git push -u origin main
   o
   git push -u origin master

Cómo evitar subir lo que no debe
Crea un archivo .gitignore en la raíz 
del proyecto con este contenido:

# NetBeans
nbproject/private/
build/
dist/
*.class

# Java
*.jar
*.war
*.ear

# Sistema
.DS_Store
Thumbs.db
